Definitions


Commercial Property

It includes shopping centers and shopping malls, medical and educational buildings, hotels and offices.


Industrial Property

It includes manufacturing buildings and property, as well as warehouses. The buildings can be used for research, production, storage, and distribution of goods.


Land Property

It includes vacant land, working farms, and ranches.


Residential Property

It includes condominiums, apartments, and landed houses.


Security Event

A security event is any observable occurrence that is relevant to security. This can include attempted attacks or lapses that expose security vulnerabilities.


Security Incident

security incident is a security event that results in damage or risk to security assets and operations.
